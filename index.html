Projects

Enverus - Request Form Manager Web App (Capstone 2022)
  • Allow authors to dynamically customize a request form template for suppliers to fill in and provide bid information.
  • Form Controls are modified to support both an Author view and a Recipient view.
  • Individual components of a form can be saved for future importing into different request forms.
  • Intuitive scoring sheet provided to allow author to score different responses.
  • Scorecard aggregates all individual scores across all metrics and provides authors the ability to flexibly increase the weight of individual categories.

Technology Used: Angular 12, Reactive Forms, HTML5, CSS3, Bootstrap 5, Kotlin, PostgreSQL, Docker, JPA Spring Boot

UofC - Veterinarian Animal Management Web App
Go to Github Repo
  • Full stack web app for veterinary health professionals and students to complete the treatment approval and scheduling process for animals.
  • Added middle tier business logic for using JSON Web Tokens as part of the authentication process to control which accounts had specific user privileges.
  • Provided a search endpoint which was capable of recognizing typos and allowed users to search for both animal names and animal species to assist with their search.
  • Image storage page allowed users to upload and delete images of an animal.
  • Comment section implemented so that all users could comment on an animal but professionals can toggle student comments off.

Technology Used: React.js, HTML5, CSS3, Bootstrap 5, MySQL, Java, Spring Boot

Movie Theater Ticket App
Go to Github Repo
  • Java Swing App allows users to sign up for a paid account and login to their paid account to receive registered account benefits.
  • Customers can search movie theater tickets based on the date, the movie theater location, and the showing time for each movie.
  • Users can cancel their ticket and will receive a credit code via email in accordance with the return policy for their respective account type.
  • App is designed with multi-threading in mind so that concurrent usage of the individual features do not result in unexpected behavior.

Technology Used: Java, Swing, MySQL
